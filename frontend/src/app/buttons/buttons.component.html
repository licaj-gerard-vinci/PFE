Comment fonctionne le formulaire ?

Le formulaire est divisé en deux parties :

Aujourd'hui : Indiquez ce que vous faites actuellement (par exemple, "Je ne regarde pas mon compteur d'eau").
Dans 2 ans : Prenez un engagement sur ce que vous allez faire dans le futur (par exemple, "Oui, je regarderai mon compteur d'eau").

Cela vous permet de planifier des actions concrètes pour améliorer vos habitudes ou vos engagements à l'avenir.

Comment fonctionne la barre de progression ?

La barre de progression montre votre avancement dans le formulaire. Plus vous complétez les étapes, plus la barre se remplit.

Par exemple : Si vous avez complété 4 étapes sur 10, la barre affichera 40%.

À quoi sert le bouton de partage ?

Le bouton de partage vous permet de partager votre progression avec vos collègues. Cela peut être utile pour collaborer sur les étapes restantes.
